# ME

Personal portfolio for **Dhruv Kamalesh Kumar** — Gen AI Engineer & Technical Lead at the Burnes Center for Social Change.

**Stack**: Next.js 16, TypeScript, Tailwind CSS v4, GSAP ScrollTrigger, Framer Motion, Vercel AI SDK, Pretext, Lenis

## Run locally

```bash
npm install
npm run dev
```

For the AI chat feature, add your OpenAI key:

```bash
cp .env.local.example .env.local
# edit .env.local with your key
```

The site works without a key — AI chat falls back to placeholder responses.

## Deploy

Push to `main` — Vercel auto-deploys. Add `OPENAI_API_KEY` in the Vercel dashboard to activate AI chat.
